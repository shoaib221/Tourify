# 🚀 Tourify

A full-stack web application for booking tours, residences, and local tour guides across different destinations around the world.

---

## ✨ Features
- Users can announce their availability for a residency vacancy and tour guide availability
- Other users can search for residences and guides according to their preferred location, time span and cost

---

## 🧰 Tech Stack

**Backend:**
- Django  
- Django REST Framework  

**Database:**
- PostgreSQL  

**Frontend:**
- Django Templates  
- HTML  
- CSS  
- JavaScript  

**Authentication:** 
- Django Authentication System
