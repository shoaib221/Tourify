# 🚀 Tourify

A full-stack web application for booking tours, residences, and local tour guides across different destinations around the world.

---

## 🌐 Live Link

👉 **[Live Demo](https://your-live-link-here.com/)**

---

## ✨ Features

- 🔐 User authentication (Login, Signup)
- 🌍 Browse destinations worldwide
- 🏨 Search and view available residences
- 🧭 Find and book local tour guides
- 📅 Booking and reservation system
- 🧑‍💼 User dashboard for managing bookings
- 🛠 Admin panel for managing destinations, residences, guides, and bookings
- 🎨 Responsive and user-friendly UI
- 🔎 Search, filter, and sort functionality
